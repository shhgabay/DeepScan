## מה זה DeepScan?
כלי חיפוש עוצמתי שנובר עמוק בתוך קבצים שבדרך כלל קשה לחפש בהם. במקום להסתפק בקבצי טקסט פשוטים, DeepScan מחלץ טקסט מפורמטים מורכבים ומאפשר לך למצוא מידע חבוי תוך שניות.

### ✨ יכולות עיקריות:
*   **חיפוש ב-Photoshop (PSD):** סורק את שכבות הטקסט בתוך קבצי עיצוב.
*   **חיפוש ב-Office:** תמיכה מלאה בקבצי Word (.docx) ו-Excel (.xlsx).
*   **חיפוש ב-PDF:** חילוץ טקסט ממסמכים דיגיטליים.
*   **חיפוש בטקסט נקי:** תמיכה ב-TXT וקבצי קוד.
